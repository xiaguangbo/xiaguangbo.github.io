## 开发环境

在 docker 里的 ubuntu 里进行开发

### 创建容器

1. 获取镜像

```terminal
docker pull ubuntu
```

3. 创建并运行容器

```terminal
docker run -itd --name ubuntu_container_rk3399 ubuntu
```

### 对 ubuntu 容器做一些预操作

1. 进入容器

```terminal
docker exec -it ubuntu_container_rk3399 bash
```

2. 更新软件

```terminal
apt-get update && upgrade
```
