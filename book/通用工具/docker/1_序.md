## 说明

docker 类似于隔离沙箱，沙箱里与外界主机互相之间无法直接交互，只能通过终端、网络等接口进行交互  
沙箱里所有改变都不会影响到外界主机，反之亦然

docker 依赖 linux 内核，如果主机不是 linux 系统可以在虚拟机里用，或者用专门为 docker 制作的软件（内部依然是虚拟机，但用起来方便）

docker for windows 会使用 windows WSL/WSL2 安装 linux 虚拟机，docker for windows 提供了一个 windows 和 docker 之间方便交互和管理的工具  
docker 内部默认路径也是在这个虚拟机里，包括下载的镜像、创建的容器等。虽然 windows 的终端可以直接使用`docker images`这类命令，但这也是被软件转到虚拟机里执行的

## 安装

搜索 docker winodws，找到并直接安装。安装的过程中会弹框说要安装一个“适用于x86_64的linux subSysteam”之类的，弹框上有链接，点过去找到并安装，然后继续安装 docker
