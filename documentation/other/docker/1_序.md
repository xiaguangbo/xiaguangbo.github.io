## 简介

在这里面用 linux 镜像就不需要手动去搞虚拟机了

## 常用命令

### cp

- 将容器里的文件复制到主机

    在主机的终端里执行

    ```
    docker cp container_name_or_id:/root/abc.txt C:\Users\16175\Downloads\
    ```

- 将容器里的文件夹复制到主机

    在主机的终端里执行

    ```
    docker cp container_name_or_id:/root/ C:\Users\16175\Downloads\
    ```

    会在 C:\Users\16175\Downloads\ 里出现一个 root 目录，里面存储着原 root 目录下的内容

方向反过来只需将 docker cp 后面的两条语句对调
