命令行文件编辑器

## 安装

- linux：包名为 vim

## 使用

打开文件：vim file_directory  
如果文件不存在会自动创建。首先会进入命令模式

命令模式用于剪切、复制、粘贴、撤销、恢复等功能  
编辑模式用于编辑文本  
末行模式用于保存文件、退出等

- 模式切换

    切换到编辑或末行模式都要在命令模式里进行，进入命令模式按 esc

    - 进入编辑模式：i
    - 进入末行模式：:

## 功能

- 末行模式
    
    强制执行就在后面加个!字符
    
    - 保存：:w
    - 保存并退出：:wq
    - 不保存并退出：:q

- 命令模式
    - 全部删除：ggdG
