## e5v3 鸡血 bios

```
最好是自己手动制作，防止因为 bios 的 rom id 不一致导致主板点不亮

信息
    https://placaschinas.com/como/s3turbotool/
    http://xeonlive.ru/instruktsii/unlock-turbo-boost-s-pomoshchyu-s3turbotool

软件 S3TurboTool
    https://disk.yandex.ru/d/IA0OR3QmAKtYTA?w=1
```

```
S3TurboTool
    ... fullbuckup
        提取 bios
    
    MMtool5
        修改 bios cpu 微码
        如果界面显示不全，则使用键盘的方向键来上下选择和空格键来确认
    
    选项 3
        修改 cpu 电压、北桥频率，生成 ffs 文件，文件在 S3TurboTool/S3TurboHack/
    
    UEFITool
        替换 ffs 文件

    AMIBCP
        修改 bios cpu 节能功能
```

```
e5 2630v3 8c16t
    电压
        Core      -60
        Cache     -60
        SyS agent -50

    SVID/FVR
        勾选
```
