## 简介

简单、明了、安全、快速

程序可分为“必须连续运行”和“可独立运行”两部分，可独立运行的部分可优化成多任务，可再优化成多核运行。对程序而言串行和并行没有本质的区别，结果是一样的。多任务时，某个任务需要等待其他程序传递一些信息时该任务才会原地打转，但并不影响整体，因为多任务是轮转运行或并行的。  
cpu 和 fpga 的程序对于连续执行是一样的。面对多任务，单核 cpu 只能轮转，而多核 cpu 可以并行，fpga 的多任务可以是多个功能电路块。  
也就是说，通用语言完全是可行的。虽然面对不同的使用场景会出现不同的使用方法，比如面对属性描述的场景时就只有变量了。

从语法逻辑来看，目前的编程语言可分为三类，分别是顺序执行（C/C++、C#、Rust、JavaScript、Python、Shell）、顺序和并行执行并存（Verilog HDL）、属性描述（JSON、HTML、CSS）。显然，需要三种语言以满足需求，但他们的语法并不是大相径庭的，比如 JSON 是 JavaScript 的子集，JavaScript 很像 C，还比如 Verilog HDL 是用于 FPGA 的语言，各逻辑语句、功能块之间可以连接以并行运行，自动触发，无需像顺序执行的语言那样通过通知或手动调用，但其语法依然很像 C，只是添加了更多的符合以表示更多的功能。可以有一个完整的虚拟语法然后分出这三种类型的语言

规则简单明了才能通用且快速，拒绝概念混杂，拒绝概念漏洞才能安全

参考特性有内存引用计数式释放、并发、模块、导入导出等
