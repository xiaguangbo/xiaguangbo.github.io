简单、无限扩展

运行于其他物理传输协议之上，将其作为透明传输介质使用，以扩展出无限多的功能

依赖其他物理传输协议的原因是不同的物理介质传输数据的方式也不同，因此使用的物理传输协议也就要满足传输数据的要求。不同的使用场景也会使用不同的物理传输协议以满足场景要求。因此在这些因素下出现的传输协议五花八门，各不兼容，比如包括电气上的和软件支持上的兼容性。所以要一个虚拟设备来保证软件支持的一致性

数据是一包一包的，由 包大小 + 数据 + 校验码 组成，一个包最大为 255 字节
包大小 占 1 字节，包大小 表示本包的大小，值的范围是 3~255。数据 占的字节数是 1~253。校验码 占 1 字节，值是 包大小 和 数据 部分的 crc8 校验码
