从语法逻辑来看，编程语言分为三类，分别是顺序执行（C/C++、C#、Rust、JavaScript、Python、Shell）、顺序和并行执行并存（Verilog HDL）、属性描述（JSON、HTML、CSS）。显然，需要三种语言以满足需求，但他们的语法并不是大相径庭的，比如 JSON 是 JavaScript 的子集，JavaScript 很像 C，还比如 Verilog HDL 是用于 FPGA 的语言，各逻辑语句、功能块之间可以连接以并行运行，自动触发，无需像顺序执行的语言那样通过通知或手动调用，但其语法依然很像 C，只是添加了更多的符合以表示更多的功能。可以有一个完整的虚拟语法然后分出这三种类型的语言

规则简单明了才能通用且快速，拒绝概念混杂，拒绝概念漏洞才能安全

参考语言：C、C++、C#、Rust、JavaScript、Python、Verilog HDL 等

参考特性：内存引用计数式释放、并发、模块、导入导出等
