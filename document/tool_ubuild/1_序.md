## 简介

自动构建系统，使用 C# + .NET 7 开发

文件夹结构：
```
rockchip_ubuild
|- build
|  |- download
|  |  |- arm-trusted-firmware
|  |  |- u-boot
|  |  |- linux
|  |  |- busybox
|  |
|  |- source
|  |  |- u-boot
|  |  |- linux
|  |
|  |- cache
|  |- image
|
|- tool
|  |- wget
|  |  |- wget.ub.md
|  |
|  |- git
|  |  |- git.ub.md
|  |
|  |- make
|     |- make.ub.md
|
|- rockchip
|  |- configuration
|  |  |- firefly_rk3399.ub.md
|  |
|  |- bootload
|  |  |- arm-trusted-firmware
|  |  |  |- patch
|  |  |  |- arm-trusted-firmware.ub.md
|  |  |
|  |  |- uboot
|  |     |- patch
|  |     |- uboot.ub.md

|  |

|      |- busybox

|- ubuild

```

启动构建：
```
cd rockchip_ubuild
dotnet ubuild/main.cs rockchip/configuration/firefly_rk3399.ub.md
```

语法：
```
## value

<!-- 变量，值为字符串 -->

tool_directory = "../../../tool"

## rely

<!-- 将这些文件在当前文件的位置执行，并且会等待这些文件执行结束 -->

tool_directory + "git/git.ub.md"
tool_directory + "make/make.ub.md"

## download

<!-- 会进入 build/download 执行命令 -->

"git --help"
"cp --help"

## patch

<!-- 首先复制一份下载好的文件到 build/source/，并进入 build/source/此文件名 里执行命令，会继续下面的工作 -->

"patch --help"

## build

<!-- 会进入 build/source/arm-trusted-firmware 执行命令 -->

"make --help"

## image

<!-- 会进入 build/image 执行命令 -->

"cp --help"
```

rockchip/configuration/firefly_rk3399.ub.md：
```
## rely

"../bootload/uboot/uboot.ub.md"
"../kernel/linux/linux.ub.md"
"../rootfs/busybox/busybox.ub.md"
```

rockchip/bootload/arm-trusted-firmware/arm-trusted-firmware.ub.md：
```
## value

tool_directory = "../../../tool"

## rely

tool_directory + "git/git.ub.md"
tool_directory + "make/make.ub.md"

## download

"git clone --depth=1 https://github.com/ARM-software/arm-trusted-firmware.git"

<!-- 将文件下载到 build/download -->

## patch

<!-- 首先复制一份文件到 build/source，如果内容为空就代表不需要打包，会继续下面的工作-->

## build

"make PLAT=rk3399 CROSS_COMPILE=aarch64-linux-gnu-"

<!-- 会进入 build/source/arm-trusted-firmware，然后执行命令-->
```
