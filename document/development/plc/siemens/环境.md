TIA_Portal_STEP7_Prof_Safety_WINCC_Adv（博途） 和 PLCSIM，前者用于开发程序，后者是仿真软件，可以直接在博途打开，程序在仿真软件里运行，在虚拟机里装了才能正常用。如果第三方软件要访问仿真器，则使用 NetToPLCSim，第三方库是 snap7，C# 里就可以用。

博途要勾选“PLC/属性/常规/防护与安全/连接机制/允许来自远程对象的 PUT/GET 通信访问”，与安全有关的都不要。还有“项目/属性/保护/块编译时支持仿真”给勾上。取消勾选“添加的数据块/属性/常规/属性/优化的块访问”，不然显示不了偏移量。

在仿真软件打开之前要先打开 NetToPLCSim，等待初始化操作完成，因为 NetToPLCSim 需要 102 端口，后打开就不行了。然后启动仿真并下载程序，让仿真运行起来。点击 Add，Network IP Address：运行 PLCSim 的机器的 IP 地址，也就是虚拟机的 IP，点右边的 ... 就能选择，一般是第一个，第三方软件就连这个 IP。Plcsim IP Address：PLC 的 IP 地址，点右边的 ... 就能选择。Rack/Slot：PLC 所在的机架号和槽位号，根据给出的说明进行选择。添加完后，点 Start Server 启动服务。
